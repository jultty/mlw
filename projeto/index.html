JavaScript é uma linguagem de programação criada durante os anos 90 para que os navegadores web pudessem ter capacidades dinâmicas.

Foi originalmente desenvolvida nas empresas Netscape Communications e Sun Microsystems para ser utilizada no navegador Netscape, e depois adotada por quase todos os navegadores do mercado.

Devido à importância que a linguagem ganhou com o desenvolvimento de sistemas para Internet cada vez mais complexos, ela hoje possui implementações para diferentes ambientes além dos navegadores.

Esta página procura dar uma visão geral da história, características e aplicações do JavaScript.
